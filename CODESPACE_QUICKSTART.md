# Codespace Quickstart (Copy/Paste)

Paste this from the repo root in Codespaces:

```bash
chmod +x scripts/run_preflight_checks.sh
./scripts/run_preflight_checks.sh
```

If it prints `PASS: preflight checks complete`, your spec/data setup is valid.

## Optional: run one-liner without chmod
```bash
bash scripts/run_preflight_checks.sh
```
