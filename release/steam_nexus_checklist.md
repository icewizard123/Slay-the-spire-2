# Steam Workshop + Nexus Mods Release Checklist

## Packaging
- [ ] Include version in title and description (e.g., `v0.1.0`).
- [ ] Include dependency + load-order notes.
- [ ] Include known issues and compatibility notes.
- [ ] Add changelog section with date-stamped entries.

## Legal/IP hygiene
- [ ] Use original art/audio only.
- [ ] Do not use anime screenshots, logos, or ripped voice lines.
- [ ] Use inspiration language ("inspired by") not official franchise branding.

## Player-facing docs
- [ ] Install instructions.
- [ ] Uninstall instructions.
- [ ] Save compatibility warning for mid-run install/removal.
- [ ] FAQ: crash report/log file location.

## Launch-day QA
- [ ] New run with character completes without soft-locks.
- [ ] All starter cards/relic render and localize correctly.
- [ ] Workshop + Nexus descriptions match the shipped build.
